# galileo-motor-controller
Controller for a rotary actuator servomotor implemented for Intel Galileo Gen2.

**Author:** Levindo Gabriel Taschetto Neto.

## Components

*  Pushbutton;
*  Servomotor;
*  Display LCD with Backlight;
*  Potentiometer;

## How to use
```
$ cd src/
$ make clean
$ make
$ make run
```
